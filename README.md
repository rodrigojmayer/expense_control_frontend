This repository contains the user interface for **ExpenseControl**, a high-performance web application designed for corporate expense management and control. Built with **React 18**, **TypeScript**, and **Vite** to ensure an ultra-fast development and user experience.
## 🚀 Tech Stack

* **Core:** [React 18](https://react.dev/) + [Vite](https://vitejs.dev/) (Build tool)
* **Language:** [TypeScript](https://www.typescriptlang.org/) (Strict typing for increased robustness)
* **UI Framework:** [Material UI (MUI) v6](https://mui.com/) with **Emotion** and **tss-react**.
* **Authentication:** [Google OAuth 2.0](https://www.npmjs.com/package/@react-oauth/google) via `@react-oauth/google`.
* **Security:** Token handling with `jwt-decode`.
* **Code Quality:** ESLint v9 with hooks and react-refresh configurations.

---

## ✨ Key Features

* **Secure Authentication:** Integrated access with corporate Google accounts to protect financial data.
* **Component-Based Architecture:** MUI-based interface ensuring visual consistency and accessibility.
* **Dynamic Inventory Management:** Dedicated modals to create, edit, and delete articles or groups (categories) in real-time.
* **Guided Business Workflow:** Sequential selection system: Customer ➔ Payment Method ➔ Article Loading.
* **Transaction Cart:** Detailed summary of total expenses with a preview option before final export.
* **Google Sheets Integration:** Interface designed to trigger automatic exports to spreadsheet history, organized by customer.
